### 문제 소개
[피보나치_함수](https://www.acmicpc.net/problem/1003)

### 입/출력 분석
- 모든 입/출력 경우의 수를 나열하시오.

input 

t
X1
X2
...
Xt

output

f(X1)
...
f(Xt)

### 풀이
- 풀이 과정 (풀이의 논리가 담긴 글)

풀이 과정 수식화 - 손으로 풀기

f(0) -> [1 ,0]
f(1) -> [0 ,1]
f(2) -> [1, 1]
f(3) -> [1, 2]
...
f(n) -> f(n - 1) + f(n - 2)

DP로 풀면 되겠다!

- 풀이의 중심이 되는 알고리즘이 무엇인가?
DP

- 왜 이 알고리즘을 선택했는가?
점화식으로 표현 가능해서.

- 시간복잡도를 계산하시오.
O(N)

- 공간복잡도를 계산하시오.
O(N)

- special case가 있는가?

### 슈도 코드 작성

입력 -> input[]

m = max(input)
dp = [(0, 0); m]

dp[0] = (1, 0)
dp[1] = (0, 1)

for i in (2..=m) {
	dp[i] = dp[i - 2] + dp[i - 1];
}

for i in input {
	println(dp[i])
}
